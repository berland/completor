# Setup

CompletorÂ® is available through the Komodo distribution system, it is possible for you to have it available on your machine by activating a stable Komodo release:
```shell
source /prog/res/komodo/stable/enable
```
You can also start it by cloning the repository:
```shell
git clone https://github.com/equinor/completor.git
cd completor
pip install -e ".[tests,docs]"
```
and to ensure it runs as intended, confirm it with:
```shell
completor --help
```
and
```shell
pytest
```
This should run for some minutes without failures.
